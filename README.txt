This is a csgo webcrawler

To update information:
1) Run scrape_id.py (to create csgo_skin_ids.csv)
2) Run scrape.py (to apply csgo_skin_ids.csv)
3) All info in csgo_skins.csv (generated from scrape.py)
4) Run sort_skins.py (to sort out csgo_skins.py)
5) csgo_skins_sorted.csv is created
6) Import csgo_skins_sorted.csv into Google Spreadsheet and format
